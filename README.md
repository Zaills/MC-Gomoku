# MC-Gomoku
Minecraft mod to Connect to Gomoku's Server

Work with https://github.com/Cmasserann/Gomoku
Run the server before launching the client

`/go` create a vs AI
`/go create local` to create a player vs player local game
`/go create remote` to create a player vs player remotely game
`/go join [code]` to join a player vs player game
`/go giveUp` to giveUp the game

closing the client will also make you gave up
